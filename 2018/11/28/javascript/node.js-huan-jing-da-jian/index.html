安装 Node.js

Debian / Ubuntu:

apt-get install nodejs

通过查看版本号来验证 Node.js 是否安装成功

node -v

安装 NPM

查看当前镜像地址

npm get registry

设置镜像地址

npm config set registry http://registry.npm.taobao.org/

版本升级

NPM 版本升级

npm install -g npm

Node.js 版本升级

通过 n 模块来进行更新

npm install -g n

安装最新 LTS 版本

n stable
